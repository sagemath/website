Citing {{ sage }}

If you use {{ sage }} in a book, paper, website, etc., please contact us about details of the publication, e.g. where it is published, provide a link to your publication. Please reference {{ sage }} as described here. The preferred way to submit publication details is to provide them as a pull request on the dedicated GitHub repository to update the file bibliography-sage.bib.
Also, be sure to find out which components of {{ sage }}, e.g. NumPy, PARI, GAP, Sage-Combinat, that your calculation uses and properly attribute those systems. If you are unsure, ask on the sage-support mailing list. Similarly, consider finding out who wrote the {{ sage }} code you are using and acknowledge them explicitly as well.
If your work (book, paper, website, etc.) is supported by the {{ sage }} cluster, please contact us. Please also acknowledge the grant that supports the purchase of the {{ sage }} cluster by adding the following acknowledgment to your work: National Science Foundation Grant No. DMS-0821725.
